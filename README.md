# 视频会议系统

本项目是一个基于实时通信技术的视频会议系统，采用前后端分离的 B/S 架构，使用 WebRTC 实现浏览器端音视频通信，使用 WebSocket 实现信令交换，并结合 STUN/TURN 服务支持 NAT 穿透。

## 项目简介

系统面向远程办公、在线教育和远程协作等场景，支持用户通过浏览器创建会议、加入会议、进行实时音视频交流，并可在后续扩展屏幕共享、聊天、会议录制等功能。

## 技术栈

- 前端：Vue 3
- 后端：Go + Gin
- 实时通信：WebRTC / WebSocket
- 数据库：MySQL
- 穿透服务：STUN / TURN
- 版本管理：Git
